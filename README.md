##  Projets Principaux

###  Cafeterie - Plateforme de Gestion Collaborative
**Application de centralisation des achats et événements en formation**

Une plateforme complète permettant de gérer collectivement les achats de boissons chaudes et repas, avec système d'organisation d'événements entre utilisateurs.

**Architecture :** Vue.js 3 + Node.js/Express + MongoDB  
**Fonctionnalités :**
- Gestion de commandes et historique d'achats
- Système d'événements collaboratifs
- Authentification utilisateurs et profils
- Dashboard de suivi en temps réel

**Tech Stack :** Vue.js 3.4, Pinia, Vue Router, Express, MongoDB, Mongoose

 [Front-End Repository](https://github.com/Gaetan1303/Cafeterie) | [Back-End Repository](https://github.com/Gaetan1303/Cafeterie-Back)

---

###  GameMaster Legend of the 5 Rings - Plateforme de Jeu de Rôle
**Application collaborative pour maîtres de jeu et joueurs**

Système complet de gestion de parties de jeu de rôle dans l'univers Legend of the 5 Rings, avec interface GameMaster et interface joueur séparées.

**Architecture :** Application multi-repositories avec partage de backend

**GM_L5R (Interface GameMaster) :**
- Création et gestion de rooms de jeu
- Simulation de parties en temps réel
- Gestion multi-joueurs synchronisée
- Administration de sessions

**JDR-test (Interface Joueur) :**
- Gestion de personnages multiples
- Connexion aux rooms créées par le GM
- Interface joueur immersive
- Synchronisation temps réel avec le backend

**Tech Stack :** TypeScript, Node.js/Express, MongoDB, Architecture temps réel

 [GameMaster Repository](https://github.com/Gaetan1303/GM_L5R) | [Joueur Repository](https://github.com/Gaetan1303/JDR-test)

###  Aether-Engine - Backend Go Temps Réel
**Backend Go pour la synchronisation temps réel et la logique métier avancée**

Moteur backend en Go conçu pour la synchronisation temps réel et le traitement de logique métier complexe, conteneurisé avec Docker.

**Architecture :** Go + Docker  
**Fonctionnalités :**
- Synchronisation temps réel
- Logique métier avancée
- API exposée sur le port 8080
- Pipeline CI/CD automatisé

**Tech Stack :** Go, Docker

 [Repository](https://github.com/Gaetan1303/Aether-Engine)

###  MMO-RPG - Template RPG Multijoueur
**Application RPG multijoueur avec Spring Boot, WebSocket et PostgreSQL**

Template de jeu RPG multijoueur complet avec authentification JWT, communication temps réel via WebSocket/STOMP, base de données PostgreSQL et orchestration Docker Compose.

**Architecture :** Spring Boot API + PostgreSQL + Mercure (SSE) + Caddy (reverse proxy)  
**Fonctionnalités :**
- Authentification JWT
- Communication temps réel WebSocket/STOMP
- Gestion multi-joueurs
- Orchestration complète via Docker Compose

**Tech Stack :** Java, Spring Boot, PostgreSQL, WebSocket/STOMP, Mercure, Docker Compose

 [Repository](https://github.com/Gaetan1303/MMO-RPG)
